jQuery瀑布流

發表於 2015-02-25   |   分類於 jquery

跟教程敲了一下jQuery瀑布流，整理一下思路：

html+css部分

每一个<img>有两层的<div>嵌套，外层的添加一个box类，内层的添加一个content类。每张图片的宽度固定为190px，高度设为auto。

js部分

首先理清思路，分为两部分，先写图片布局方式(imgLocation() 函数)，再写滚动加载(scrollside() 函数)。

定位图片 imgLocation()

  1. 首先计算出一行可以放置几张图片
  2. 创建一个数组，初始化值设置为第一行各图片高度（each循环）
  3. 获取最低高度及最低高度索引
  4. 将当前循环的对象绝对定位，top设为第三步获得的最低高度，left设为最低高度索引对应的对象的left值。
  5. 重置当前最低高度索引对应的数组元素值为原来的值加上当前循环对象的高度。然后返回第二步，重复执行循环语句。
    在主函数中主需要调用就行。

滚动加载 scrollside()

lastboxHeight：为最后一张图片的距顶端高度
documentHeight：文档可见高度
scrollHeight：滚动条距顶端高度
当满足条件：lastboxHeight < scrollHeight + documentHeight的时候，返回true，执行滚动加载的函数。
